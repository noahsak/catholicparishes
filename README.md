# Catholic Parishes

This GitHub repository is home to the code and database behind the Catholic Parishes website. Learn more about Catholic Parishes [here](https://www.catholicparishes.org/about)

## Catholic Parishes Database

The database behind Catholic Parishes is located in public/data/ if you want to take a look or use it for whatever reason. Again, I try my best to have it be as accurate as possible, but I am only human and prone to mistakes.
